Gmail 電商取貨提醒自動化

自動化GmailGoogle Calendar電商取貨提醒生活助手通知解析

這個工作流程自動監聽 Gmail 中的電商取貨通知信件，並在 Google Calendar 中建立智能取貨提醒。系統能夠識別不同電商平台（一般電商、賣貨便）的到店通知，解析出門市資訊，自動建立個人化的行事曆提醒，避免忘記取貨。 主要功能： 自動信件監聽：每小時檢查 Gmail 中的取貨通知 多平台支援：支援一般電商和賣貨便的取貨通知 智能資訊解析：自動提取門市名稱和取貨資訊 個人化提醒：建立包含門市資訊的行事曆事件 雙重提醒：設定 20 分鐘和 40 分鐘前提醒

設置說明

預先設定：需要設定 Gmail 和 Google Calendar API 存取權限

  1. Gmail API 設定: 設定 Gmail OAuth2 認證
    • 在 Google Cloud Console 啟用 Gmail API
    • 建立 OAuth2 憑證
    • 在 n8n 中設定 Gmail OAuth2 認證
    • 測試 Gmail 連線
  2. Google Calendar API 設定: 設定 Google Calendar OAuth2 認證
    • 在 Google Cloud Console 啟用 Calendar API
    • 使用相同的 OAuth2 憑證或建立新的
    • 在 n8n 中設定 Google Calendar OAuth2 認證
    • 選擇目標行事曆（建議建立專用行事曆）
    • 測試行事曆連線
  3. 信件過濾調整: 根據個人使用的電商平台調整搜尋條件
    • 修改 Gmail 搜尋關鍵字以符合實際收到的信件
    • 調整正規表達式以正確解析門市名稱
    • 測試信件解析功能
    • 設定適合的監聽頻率
