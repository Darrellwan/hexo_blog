Line 收據AI辨識自動記帳

自動化AI記帳拆帳Line收據辨識Google Sheets

這個工作流程透過 Line 機器人接收使用者上傳的日本收據圖片，使用 AI 自動辨識收據上的金額、品項和店家資訊，並提供拆帳功能，可以在多人之間平均分攤費用。系統會將所有資料記錄到 Google 試算表中，方便後續查詢和管理。 主要功能： 收據辨識：使用 OpenAI 模型辨識日文收據上的文字和金額 資料擷取：自動提取品項、金額、店家和日期等資訊 自動記帳：所有消費記錄自動儲存到 Google 試算表 拆帳功能：可選擇單人記帳或多人拆帳，自動計算每人應付金額

功能特色
  • 透過 Line 接收圖片或直接輸入資訊
  • 使用 OpenAI 辨識日文收據上的關鍵資訊
  • 自動提取品項、金額、店家和日期資訊
  • 提供單人記帳和多人拆帳選項
  • 自動儲存所有交易記錄到 Google 試算表
  • 支援重試機制，處理辨識錯誤情況
設置說明

預先設定：需要設定 Line Messaging API、OpenAI API 和 Google Sheets 的 API 存取權限

  1. Line Bot 設定: 設定 Line 機器人和 Webhook
    • 建立 Line Developer 帳號
    • 設定 Messaging API
    • 取得 Channel Access Token
    • 設定 Webhook URL
  2. OpenAI 設定: 設定 OpenAI API 用於收據辨識
    • 取得 OpenAI API 金鑰
    • 選擇適當的模型（推薦使用支援圖像辨識的模型）
  3. Google Sheets 設定: 準備用於記錄交易資料的試算表
    • 建立 Google 試算表
    • 設定「expense」和「user」兩個工作表
    • 設定適當的欄位格式
