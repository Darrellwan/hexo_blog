n8n DataTables 完整功能示範

DataTables表單收集CRUD 操作資料管理n8n 教學

這個工作流程展示 n8n DataTables 的完整功能，包含表單資料收集和 7 種核心 CRUD 操作演示。透過實際的用戶評價表單收集數據（email、評分、訂閱選擇、生日），並展示 Insert、Get、Update、Upsert、Delete、If Row Exists 等所有 DataTables 操作方式。 主要功能： 表單觸發：使用 n8n Form Trigger 收集用戶評價和資料 數據插入：自動將表單資料寫入 DataTables 條件查詢：根據 email 等條件查詢特定資料 存在判斷：檢查資料是否存在的邏輯判斷 數據更新：更新現有資料的評分等欄位 智能插入：Upsert 操作實現插入或更新的自動判斷 資料清理：根據條件刪除不需要的測試資料

設置說明

預先設定：需要 n8n 1.113.0 以上版本支援 DataTables Beta 功能

  1. 版本確認: 確保 n8n 版本支援 DataTables
    • 檢查 n8n 版本為 1.113.0 或更高
    • 確認左側選單中有 Data tables 選項
    • DataTables 目前為 Beta 功能，建議先在測試環境使用
  2. 建立 DataTable: 建立名為 form_users 的資料表
    • 點擊 Data tables → Create Data table
    • 表格名稱：form_users
    • 新增欄位：email (String)、review_stars (Number)、subscribe (Boolean)、birthday (Date)
    • 確保欄位類型正確對應表單資料格式
  3. 匯入並測試工作流程: 匯入模板並進行功能測試
    • 匯入 workflow JSON 檔案
    • 檢查 DataTable ID 是否正確對應到 form_users
    • 測試表單提交功能：填寫測試資料並送出
    • 逐一測試每個 DataTables 操作節點
    • 觀察每個操作的執行結果和資料變化
    • 注意：Delete 操作會真的刪除資料，測試時請小心
