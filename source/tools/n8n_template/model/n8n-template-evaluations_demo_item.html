n8n Evaluations 功能示範：AI 項目分類

自動化AI評估分類Google SheetsOpenRouter測試範例

這個工作流程展示 n8n 的 Evaluations 功能，使用 AI 模型自動分類項目為「動物」或「水果」。系統從 Google Sheets 讀取測試資料，透過 AI Agent 進行分類，並將結果寫回試算表進行評估和比較。這是一個完整的 AI 模型評估範例，適合用於測試不同 AI 模型的分類準確度。 主要功能： 資料集整合：從 Google Sheets 讀取測試項目 AI 分類：使用 OpenRouter 的 Gemma 模型進行項目分類 結果評估：將 AI 分類結果與標準答案進行比較 自動化測試：批量處理多個測試項目 準確度分析：評估 AI 模型的分類準確度

設置說明

預先設定：需要設定 Google Sheets API 和 OpenRouter API 存取權限

  1. Google Sheets 設定: 準備測試資料集和結果記錄表
    • 複製範例 Google Sheets 模板
    • 設定 OAuth 2.0 認證
    • 確認工作表包含 item 和 category 欄位
    • 授予讀寫權限
  2. OpenRouter 設定: 設定 OpenRouter API 用於 AI 模型存取
    • 註冊 OpenRouter 帳號
    • 取得 API 金鑰
    • 選擇適當的模型（範例使用 google/gemma-3-1b-it:free）
    • 測試 API 連接
  3. 測試資料準備: 準備用於評估的測試項目
    • 在 Google Sheets 中新增測試項目
    • 確保每個項目都有正確的分類標籤
    • 建議準備至少 10-20 個測試項目
    • 包含明確和模糊的分類案例
