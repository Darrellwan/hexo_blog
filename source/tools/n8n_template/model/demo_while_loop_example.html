While True 迴圈範例：分頁資料處理

自動化迴圈分頁API教學範例

這個工作流程展示如何在 n8n 中實現 while 迴圈功能，特別適用於處理分頁 API 資料。工作流程模擬 API 回傳分頁資料的情境，並透過條件判斷和迴圈結構來處理所有分頁。 主要功能： 動態參數設定：可自訂起始位置、每頁資料量和最大執行次數 模擬 API 回傳：根據參數動態生成資料陣列 條件判斷：檢測是否已到達最後一頁或達到最大執行次數 迴圈實現：透過節點連接實現 while 迴圈功能
