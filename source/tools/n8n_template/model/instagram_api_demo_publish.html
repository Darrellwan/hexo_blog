Instagram API 貼文發佈範例

自動化社交媒體InstagramAPI貼文發佈範例

這個工作流程展示如何使用 Instagram Graph API 發佈圖片貼文。流程首先驗證 Access Token 是否有效，接著建立內容容器（container），最後將容器發佈為正式貼文。適合用於自動化內容發佈和社群媒體排程。 主要功能： Token 驗證：確認 API Access Token 有效性 內容容器建立：上傳圖片 URL 並準備貼文內容 貼文發佈：將準備好的容器正式發佈至 Instagram

設置說明

預先設定：需要設定 Instagram Business 帳號和 Facebook Developer 應用程式

  1. Facebook Developer 設定: 設定 Facebook 應用程式並取得 Instagram API Access Token
    • 建立 Facebook 應用程式
    • 設定 Instagram Graph API
    • 取得長期 Access Token
    • 設定應用程式權限
  2. Instagram 帳號設定: 確認 Instagram 帳號已轉為商業帳號並連結至 Facebook 專頁
    • 轉換為商業帳號
    • 連結 Facebook 專頁
    • 開啟內容發佈權限
