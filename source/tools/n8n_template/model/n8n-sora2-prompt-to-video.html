OpenAI Sora-2 文字轉影片自動生成

AI影片生成OpenAISora自動化Google Drive文字轉影片

這個工作流程使用 OpenAI 最新的 Sora-2 API 將文字提示詞轉換成高品質影片。系統會自動處理影片生成請求、監控生成狀態、下載完成的影片，並將成品自動上傳到 Google Drive 儲存。支援不同的影片長度（4/8/12 秒）和解析度選項，適合用於快速生成社群媒體內容或影片原型測試。 主要功能： 影片生成：使用 Sora-2 或 Sora-2-Pro 模型根據文字提示生成影片 狀態監控：自動檢查影片生成進度，直到完成為止 自動儲存：將生成的影片自動上傳到 Google Drive 彈性設定：支援多種影片長度和解析度選項

設置說明

預先設定：需要設定 OpenAI API 和 Google Drive 的 API 存取權限

  1. OpenAI 設定: 設定 OpenAI API 用於 Sora-2 影片生成
    • 取得 OpenAI API 金鑰（需要有 Sora API 存取權限）
    • 確認帳戶已啟用 Sora-2 或 Sora-2-Pro 模型
    • 了解 Sora API 定價：sora-2 為每秒 $0.10，sora-2-pro 依解析度為 $0.30-$0.50/秒
    • 準備影片生成的文字提示詞（prompt）
  2. 調整提示詞和參數: 在 HTTP Request-prompt to video 節點中設定影片生成參數
    • 修改 prompt 為你想要的影片描述
    • 選擇 model：sora-2（基礎版）或 sora-2-pro（進階版）
    • 設定 seconds：4、8 或 12 秒
    • 選擇 size：720x1280（直向）或 1280x720（橫向）
    • 參考 Sticky Note 中的完整定價表進行成本評估
  3. Google Drive 設定: 準備用於儲存生成影片的雲端資料夾
    • 在 Google Drive 建立專用資料夾用於存放生成的影片
    • 設定適當的存取權限
    • 複製資料夾 ID（從網址列取得）
    • 在 Upload file 節點中更新 folderId 參數
