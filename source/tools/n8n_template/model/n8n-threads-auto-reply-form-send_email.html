Threads 回覆小精靈 - 表單自動寄送模板

ThreadsGmail自動化行銷Social

這個工作流程展示如何結合 Threads 貼文回覆、表單搜集和自動寄送 Email 的完整自動化流程。適合用於活動報名、模板發送、資料收集等需要多渠道整合的自動化場景。 主要功能： 表單資料搜集：使用 Form Trigger 收集使用者的 email 和 Threads 帳號 Threads 回覆監控：每小時自動抓取指定貼文下的回覆，並儲存到 Google Sheets 智能資料比對：根據 Threads 帳號合併表單資料和回覆資料 關鍵字過濾：檢查回覆內容是否包含指定關鍵字（如「Line」或「line」） 自動郵件寄送：符合條件的使用者會收到精美的 HTML 模板郵件，內含下載連結

功能特色
  • 表單搜集使用者 email 和 Threads 帳號，自動存入 Google Sheets
  • 每小時定時抓取 Threads 貼文回覆（支援分頁），並儲存至 Sheet
  • 讀取表單資料並根據 Threads 帳號（username）智能合併回覆內容
  • 過濾已寄送名單，並檢查回覆是否包含指定關鍵字（如 Line）
  • 逐筆寄送精美 HTML 郵件（含下載連結與社群資訊）
  • 更新 Google Sheets 標記寄送狀態與時間，避免重複寄送
設置說明

預先設定：需要 Threads API 權限、Gmail 帳號、Google Sheets API 存取權限

  1. Threads API 設定: 在 Meta Developer 取得 Access Token 並設定認證
    • 前往 Meta Developer 建立應用程式並新增 Threads 產品
    • 取得 Access Token 並在 n8n 設定 HTTP Query Auth Credential
    • 取得要監控的 Threads 貼文 ID
  2. Google Sheets 設定: 建立 Sheet 並設定欄位結構
    • 建立「email」工作表，欄位：email, threads, subscribe, submittedAt, sent, sent_at
    • 建立「回覆」工作表，欄位：id, text, username, sent_email
    • 在 n8n 設定 Google Sheets OAuth2 Credential 並測試連線
  3. Gmail 設定: 設定 OAuth2 並自訂郵件內容
    • 在 n8n 新增 Gmail OAuth2 Credential 並完成授權
    • 修改 HTML 郵件模板（下載連結、社群連結、文案）
  4. 調整參數設定: 根據實際需求調整關鍵參數
    • 更新 Threads 貼文 ID 和關鍵字過濾條件
    • 調整執行頻率（預設每小時 25 分）
    • 更新 Google Sheets 的 documentId 和 sheetName
