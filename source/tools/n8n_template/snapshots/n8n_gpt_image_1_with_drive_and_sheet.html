ChatGPT Image-1 圖像生成與自動儲存

這個工作流程使用 ChatGPT Image-1 模型生成高品質圖像，並自動將圖像儲存到 Google Drive 和記錄到 Google Sheets 中。系統接收使用者的文字提示，透過 OpenAI API 生成多張圖片，然後將這些圖片儲存到指定的 Google Drive 資料夾，同時在 Google Sheets 建立記錄，包含提示文字、圖片連結和消耗的 token 統計資訊。 主要功能： 圖像生成：使用最新的 GPT-Image-1 模型根據文字提示生成高品質圖像 自動儲存：將生成的圖片自動儲存到 Google Drive 指定資料夾 數據追蹤：記錄所有圖片生成記錄，包含提示文字和圖片連結 Token 用量追蹤：自動計算並記錄每次生成的 token 用量和估計成本
