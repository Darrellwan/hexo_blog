Meta Webhook Challenge 驗證回應

這個工作流程用於處理 Meta（Facebook）平台的 Webhook 驗證挑戰。當設定 Facebook/Instagram Webhook 時，Meta 會發送一個驗證請求，包含 hub.challenge 參數，此工作流程會自動回應該挑戰值以完成 Webhook 驗證程序。這是設定 Meta API 整合時的必要步驟。 主要功能： Webhook 驗證：接收 Meta 平台的驗證請求 自動回應：提取 hub.challenge 參數並回傳 簡化設定：快速完成 Meta Webhook 驗證流程 標準合規：符合 Meta 官方 Webhook 驗證要求
