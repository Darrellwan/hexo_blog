房屋資訊爬蟲到 Google Sheet

這個工作流程展示如何使用 n8n 的 HTTP Request 節點進行房屋網站爬蟲，從房君子網站抓取房屋資訊並自動儲存到 Google Sheet。此範例示範了完整的網頁爬蟲流程，包括 HTML 解析、資料結構化處理，以及與 Google Sheets 的整合。 主要功能： 網頁資料請求：使用 HTTP Request 節點抓取房屋網站內容 HTML 解析：透過 HTML 節點解析房屋列表和詳細資訊 資料分割：使用 Split Out 節點將列表資料分離成個別項目 結構化處理：提取房屋標題、社區、地址、坪數、格局等詳細資訊 Google Sheets 整合：自動將整理後的資料儲存到指定試算表

標籤: 網頁爬蟲, 房屋資訊, HTML解析, Google Sheets, 資料收集, 自動化, 不動產
