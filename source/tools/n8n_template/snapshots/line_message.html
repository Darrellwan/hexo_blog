Line Message API 完整教學範例

這個工作流程提供 Line Messaging API 的完整教學範例，包含自動回覆和主動推送兩種核心功能。當使用者透過 Line 傳送訊息時，系統會自動判斷訊息類型並回覆相同內容；同時也可以透過 Line UID 主動發送訊息給特定使用者。工作流程包含豐富的 Webhook 範例，展示如何處理文字、圖片、聲音、貼圖、表情符號和位置等不同類型的訊息。 主要功能： 自動回覆：使用 replyToken 接收並回覆使用者訊息 主動推送：透過 Line UID 或群組 ID 主動發送訊息 訊息類型支援：文字、圖片、聲音、貼圖、表情符號、位置訊息 Webhook 範例：包含完整的 JSON 格式範例供參考 條件判斷：智能識別不同類型的 Line 事件
