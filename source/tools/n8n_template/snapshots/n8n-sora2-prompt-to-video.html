OpenAI Sora-2 文字轉影片自動生成

這個工作流程使用 OpenAI 最新的 Sora-2 API 將文字提示詞轉換成高品質影片。系統會自動處理影片生成請求、監控生成狀態、下載完成的影片，並將成品自動上傳到 Google Drive 儲存。支援不同的影片長度（4/8/12 秒）和解析度選項，適合用於快速生成社群媒體內容或影片原型測試。 主要功能： 影片生成：使用 Sora-2 或 Sora-2-Pro 模型根據文字提示生成影片 狀態監控：自動檢查影片生成進度，直到完成為止 自動儲存：將生成的影片自動上傳到 Google Drive 彈性設定：支援多種影片長度和解析度選項
