Threads 數據自動抓取與分析

這個工作流程自動化抓取 Threads 平台的帳號數據，包括貼文互動數據和用戶追蹤者統計。系統會定期從 Threads API 獲取資料，並將其儲存到 Google 試算表中，方便後續分析和視覺化。 主要功能： 貼文數據抓取：獲取所有發布貼文的詳細資訊 互動數據分析：收集每篇貼文的觀看次數、讚好、回覆、轉發等數據 用戶洞察：追蹤粉絲增長和每日互動統計 Google 試算表整合：將所有數據自動儲存到結構化表格中

標籤: 自動化, 社交媒體, 數據分析, Threads, Google Sheets
