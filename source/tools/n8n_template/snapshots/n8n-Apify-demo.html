Apify 撈 Youtube 影片資訊 Demo

這個工作流程展示如何使用 Apify 平台進行網頁爬蟲自動化，透過 n8n 整合 Apify Actors 來爬取 YouTube 搜尋結果。此模板示範了完整的爬蟲流程，包括任務啟動、狀態監控、資料提取等功能，適合需要大規模網頁資料收集的場景。 主要功能： 智能任務管理：自動啟動 Apify Actor 並監控執行狀態 等待機制：透過 Wait 節點處理異步爬蟲任務 狀態檢查：使用條件判斷確保任務成功完成 資料提取：自動從 Apify Dataset 取得爬蟲結果 錯誤處理：具備重試機制和狀態驗證
